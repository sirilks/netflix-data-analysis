# Netflix Data Analysis 📊

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset to uncover insights about content distribution, genres, release trends, and more.

---

## 📁 Dataset
Netflix Titles Dataset containing information about movies and TV shows available on Netflix.

Features include:
- Title
- Type (Movie / TV Show)
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Genre

---

## ⚙️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Scikit-Learn
- Jupyter Notebook

---

## 📊 Project Workflow

1. Data Loading
2. Data Cleaning
3. Handling Missing Values
4. Exploratory Data Analysis
5. Data Visualization
6. Insights Extraction

---

## 🔎 Key Analysis

- Distribution of Movies vs TV Shows
- Content release trend over the years
- Top countries producing Netflix content
- Most common genres
- Missing value analysis
